# Stride — HR Admin Guide (G4 / RAV-99)

**Audience:** HR managers and system administrators after go-live.

---

## First-week checklist

1. Organization profile — legal name, logo, timezone.
2. Departments and job titles.
3. Employee CSV import — validate KRA PIN, NSSF, SHIF, bank.
4. Managers and leave approvers assigned.
5. Leave types and opening balances.
6. ESS invites — `/ess/login`.
7. Test payroll in draft before first live period.

---

## Modules & access

Menu visibility follows **subscription entitlements**. Missing module → contact account manager.

Roles: Admin (settings), HR (people/leave), Finance (payroll/accounts), ESS (self-service only).
